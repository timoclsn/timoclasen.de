## Running Locally

```bash
$ git clone https://github.com/timoclsn/timoclasen.de.git
$ cd timoclasen.de
$ npm install
$ npm run dev
```

Create a `.env.local` file similar to [`.env.example`](https://github.com/timoclsn/timoclasen.de/blob/main/.env.example).
